### Descriptions of the variables from “TidyData.txt”

***NOTES:***
The _561-feature vector_ for each pattern was obtained the following way:
* signals were obtained in multiple time windows
* _linear acceleration_ and _angular velocity_ signals were measured along the XYZ-axes using the cell phone’s built-in accelerometer and gyroscope, respectively
* _UNITS_: linear acceleration = gravity acceleration (g), angular velocity = radians/second
* linear acceleration signal was separated into _body_ and _gravity components_ using low-pass Butterworth filter
* _magnitudes_ of all 3D signals were calculated using Euclidean norm
* _jerk signals_ were obtained by deriving body linear acceleration and angular velocity in time 
* _Fast Fourier Transform (FFT)_ was applied to some signals producing frequency domain signals
* mean, standard deviation and mean frequency were calculated for each signal across the measurement time windows
* original feature name is in the parentheses
